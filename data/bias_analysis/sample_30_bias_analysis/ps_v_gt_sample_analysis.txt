======================================================================
  Analysis & Conclusions
  Date: 2026-03-31  |  Sample: 30 articles per cohort  |  Annotators: Human + Claude (claude-haiku-4-5)
======================================================================

  1. IS BIAS DETECTABLE?
  -----------------------
  Yes, clearly -- but unevenly across datasets.

  Both annotators independently detect the same directional pattern:
  GT clone articles carry substantially more bias than PS articles.
  A Cohen's d above 0.8 is considered large. The signal is robust
  enough that two independent annotators reach the same conclusion
  without coordination.

  2. HUMAN VS CLAUDE AGREEMENT
  ------------------------------
  On GT clone (national news): strong agreement.
    - 67% exact match, 97% within +/-1 point
    - Pearson r = 0.803 -- strong correlation
    - Systematic drift near zero (+0.033): neither annotator is
      inflating scores relative to the other.
    This is publication-quality inter-rater reliability for a 0-3
    ordinal scale.

  On PS (pink slime local news): weaker agreement.
    - 53% exact match (still 100% within +/-1)
    - MAE = 0.467
    - Systematic drift = +0.333: humans consistently score PS articles
      ~1/3 point higher than Claude.
    The divergence arises from the ambiguous middle: humans call mild
    local partisan framing a 1, while Claude calls it a 0. This is a
    genuine calibration disagreement on auto-generated content that is
    structurally subtle rather than overtly partisan.

  3. DISTRIBUTION QUALITY FOR TRAINING
  --------------------------------------
  Dataset       Score distribution (0 / 1 / 2 / 3)   Assessment
  ----------    ------------------------------------   --------------------
  GT clone      37% / 23% / 30% / 10%                 Well-spread, good
                                                       class balance
  PS (human)    50% / 43% /  7% /  0%                 Skewed, no 3s
  PS (claude)   87% /  3% / 10% /  0%                 Near-degenerate

  The PS dataset is structurally limited: ~80-90% of the sample is
  auto-generated census and enrollment statistics. A model trained on
  this distribution would learn primarily to output 0.

  4. CONCLUSION
  --------------
  The bias signal IS detectable and IS consistent -- but only in the
  GT clone dataset.

  The GT clone provides a reliable training signal: strong inter-rater
  agreement (r=0.803), balanced score distribution across all four
  labels, and near-zero systematic annotator drift. It is ready for
  use as annotation ground truth.

  The PS dataset does not provide a usable training signal in its
  current form. The near-total zero dominance creates a class imbalance
  problem, and the +0.333 human-Claude gap suggests the annotation
  task is under-specified for this content type -- the rubric does not
  clearly distinguish auto-generated neutral content from subtly
  partisan local framing.

  RECOMMENDATION: Use the GT clone as the primary annotation corpus.
  The PS dataset is valuable as a comparison cohort to demonstrate that
  pink slime sources score lower on average, but it should not be used
  as training data without first filtering for substantive political
  content (e.g., yellowhammertimes, grandcanyontimes rows) and
  excluding the auto-generated statistics articles (cabusinessdaily,
  centralalamedanews, centraloctimes, chicotimes, etc.).
